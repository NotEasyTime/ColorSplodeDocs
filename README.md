# ColorSplodeDocs
An overview of the ColorSplode game architecture 

# File Structure
An overview of some important files in the repo

## actor.js
This file holds the actor class which is the base for all the paint buckets that spawn in the game

## colorspolde.js
This file is where the entry point to the game is, handling game state and the UI

## itemsobstacles.js
This file is where the item and player classes are defined

## levelsys.js
This file holds the level class and some of the level event code

## zone.js
This file holds the zone class and some zone functions
